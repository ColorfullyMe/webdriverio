Install

Either download it from github or use NPM.

{% highlight sh linenos %}npm install webdriverjs --save-dev{% endhighlight %}

To run tests on your local machine, you have to download a selenium standalone server which executes the selenium commands. You find the latest version here.

Write Tests

Run selenium server first:

{% highlight sh linenos %}java -jar path/to/your/selenium-server-standalone-2.35.0.jar{% endhighlight %}

WebdriverJS is just a simple library which enables Selenium testing for your TDD/BDD tests. It provides simple functions for navigating the web browser and checking attributes and states of DOM elements. Check out the examples.
